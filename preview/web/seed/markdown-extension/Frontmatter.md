# Frontmatter fixture

This paragraph verifies that ordinary Markdown remains editable after the YAML
frontmatter block.

## Expected behavior

- The frontmatter is preserved without becoming document text.
- YAML values retain their original structure and ordering.
- Headings, paragraphs, and lists after the frontmatter render normally.
